# TIL Day 3
## 파일 내용 되돌리기
- git restore (--staged) a.txt : 커밋 진행했을 때
- git rm --cached a.txt : 커밋 진행하지 않았을 때
- vim 화면
    1. i 입력
    2. 편집 진행
    3. 편집 완료 후 esc, :wq 입력
    4. 엔터 입력
